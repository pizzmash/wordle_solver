## Usage

基本的には雰囲気で頑張る。

### 起動
```
python main.py
```

### ファイル読み込み
Brouseを押して一行に一単語記述されたtxtファイルを読み込む。

### 次に打てばいい単語を知りたい
Searchをクリック！

### ヒントを入力したい
右下のボックスに単語を入力したあとにボックスの色を変えてApply！
